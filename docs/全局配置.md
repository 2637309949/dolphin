### viper

整个应用是使用viper `github.com/spf13/viper` 作为配置项， 配置项目一般从platform/conf/properties.go继承过来  

如何想要查看默认配置， 可以修改`app.viper: true`, 该配置会把默认配置写回你的应用配置中， 以方便修改。

```yaml
app:
  name: dolphin
  mode: release
  version: "1.0"
  host: localhost:8081
  viper: false
```
