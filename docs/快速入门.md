## 快速入手


- 首先安装Golang环境 [Go](https://golang.org/), 接着使用下面的命令安装Dolphin
```sh
$ go get -u github.com/2637309949/dolphin/cmd/dolphin
```

- 使用dolphin命令创建项目

```sh
$ mkdir helloword && cd helloword && dolphin init && dolphin build && go run main.go
```

输出:
```sh
time="2020/06/13 11:55:58" level=info msg="grpc listen on port:9081"
time="2020/06/13 11:55:58" level=info msg="http listen on port:8082"
```

    │   ├── article.auto.go

    ├── README.md
    ├── script
    │   ├── apis

    │   │   └── index.js
    │   └── request.js
    ├── sql
    │   ├── article
